# fileserve

fileserve is a simple file server.

Usage:

install

```bash
$ go install github.com/bobbyz3g/fileserve@latest
```

```bash
$ fileserve --help

Usage of ./fileserve:
  -bind string
        bind address (default "127.0.0.1:8080")
  -directory string
        root directory (default "./")
```
